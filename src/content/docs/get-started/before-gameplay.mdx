title: 游玩前注意事项

import Dropdown from '@components/Dropdown.astro';


- 游戏支持简体中文，不支持中文语音，部分可视化文字（字幕等）有一些英文。
- 游戏的存档一定要经常备份，有小概率会因为Bug导致存档被删除，路径为C:\Users\Administrator\AppData\Local\ReadyOrNot\Saved（AppData需要打开隐藏文件夹才会显示）备份好 saved 这个文件夹，如果被删除存档先删除 saved 文件夹再将备份的移动过去。
- 游戏为慢节奏第一人称FPS射击游戏，无跳跃、奔跑或小地图这种非拟真FPS的功能。
- 游戏内地图有一些物品具有物理性质，车辆模型只具有一定的物理性质（例如打中车辆模型的轮胎后会漏气，发出漏气的声音并且车辆模型随着轮胎的漏气慢慢下降整体高度）。
- 游戏内的电脑屏幕可以看出摩尔纹
- 游戏内枪械附件开启方式默认为 鼠标侧键（`TMB 2`）如没有鼠标侧键则需要手动更改。
- 弹药在游戏中的设定为即刻命中，子弹的伤害与距离成负相关，距离越远伤害越低。

<Dropdown title="一些游戏按键">
    **ALT+A** - 左跨步

    **ALT+D** - 右跨步

    **ALT+W** - 踮脚

    **ALT+S** - 屈膝

    **ALT+S+CTRL** - 近乎趴下

    **MMB**（鼠标中键） - 打开AI操控面板（可控制AI特警队员和人质/嫌疑人，多人模式只能控制嫌犯和人质的命令）

    **TAB** - 打开任务面板（长按TAB可以放大，任务面板可编辑队伍队伍，查看任务/任务目标等）

    **Q/E** - 左/右探头

    长按 **R** - 查看剩余弹药

    数字键 **`1/2/3/4/5/6`** - 切换 1/2/3/4/5/6 位置的装备

    长按数字键 **`3/4/5/6 + 鼠标滚轮`** - 切换位置 3/4/6 的同种类型不同装备
</Dropdown>


在穿透场景物体时，子弹的穿甲等级决定其能否穿透，但子弹还有一个属性叫做“穿透距离”，用以描述子弹穿透物体后的存能能力。

子弹小角度击中坚硬物体时会产生跳弹，触发时仅造成视觉和听觉效果，不造成伤害，也不损失命中目标的耐久。

子弹击中金属物体（铁门、钢质防弹插板等等）后有概率产生碎弹，造成小型范围伤害。受到直接伤害的人不会结算碎弹伤害，反之没有受到直接伤害的人都会结算碎弹伤害。钢质防弹板的基础碎弹概率是100%。

- 嫌疑人可能会携带不止一种武器，当武器掉落后不及时击毙/捆绑，嫌疑人会掏出另一把武器向你攻击。
